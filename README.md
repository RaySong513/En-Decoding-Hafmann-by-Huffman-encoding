## 项目简介

霍夫曼编码是一种无损数据压缩算法，通过构建最优二叉树来减少数据的存储空间。在这个项目中，使用霍夫曼编码对图像进行压缩，并通过解码还原原始图像。这里选择了《重返未来：1999》1.7版本主线剧情「今夜星光灿烂」中角色霍夫曼的图片作为测试素材。
